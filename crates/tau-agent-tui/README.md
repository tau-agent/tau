# tau-agent-tui

Ratatui-based terminal UI for the [tau](https://github.com/tau-agent/tau) LLM
agent.

Renders sessions, streams model output, and exposes tau's pickers (sessions,
tasks, themes) over a keyboard-driven interface. Used by the `tau` CLI; can
also be embedded in other binaries built on `tau-agent-lib`.
