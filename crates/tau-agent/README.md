# tau-agent

Command-line interface for the tau LLM agent. Installs the `tau` binary.

`tau` is a coding agent harness with sessions, tools, a plugin system, and a
task board for managing and parallelising work across a project. It speaks to
Anthropic, OpenAI-compatible, and other LLM providers.

## Install

```sh
cargo install tau-agent
```

Then run `tau` in a project directory. See the
[repository](https://github.com/tau-agent/tau) for configuration and usage.
